# Dawn

Dawn is a high-performance, modular finite element framework for complex, multi-physics problems. 
Designed for researchers and educator who would like to use and implement tradional and unconventional solvers, 
element formulations and materials models.

We are currently working on a first public release of Dawn in December 2025. For more information, please have a look at
[www.jorisremmers.com/software/dawn](https://www.jorisremmers.com/software/dawn).
